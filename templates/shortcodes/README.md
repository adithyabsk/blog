## Goodreads Widget

How to update the widget contents

- Go here: https://www.goodreads.com/user/edit?ref=nav_profile_settings
- Scroll down to: Grid Widget
- Shelf: `favorite-books`
- Select Sort: `Rating`, `desc`
- Number of books: 20
- Title: `Favorite Books`
- Hide Link: unchecked
- Hide Title: Checked
- Copy the snippet
